The goal is to offer a **smooth, animated, and engaging experience**, while preparing the foundation for **V2**, which will include an integrated conversational AI embodied by the avatar.

## Current Features (V1)

### Interface & UX
- Animated background based on a **custom vectorized circuit board**
- **Light effects** (halo, glow, pulse animations)
- **Animated and clickable avatar**
- **Animated navigation menu** with smooth transitions
- **Typing effect** on specific UI elements
- Fully implemented pages:  
  - Home  
  - About  
  - Projects  
  - Skills  
  - Education  
  - Experience  
  - Contact  

### Next.js Architecture
- Built with **Next.js App Router**
- Heavy use of **Client Components**
- Animations powered by **Framer Motion**
- Automatic redirect (`/` â†’ `/homepage`)
- Centralized client layout (`ClientLayout`)
